# Задание 8

**Лекция:** Стратегический менеджмент (лекция 5)  
**Источник:** [Лекция 5 СтрМен](../../lectures/lecture-5/lecture-5.md)

## Задание

1. Определить стратегию ценообразования.
2. Выбрать метод ценообразования.
3. Оценить уровень цены на товар(ы).

---

## Ответ

### 1. Стратегия ценообразования VOIS

#### 1.1 Анализ текущей ценовой позиции

VOIS позиционируется в сегменте **масс-маркет** с ценовым диапазоном 300–2 000 руб. (большинство продуктов — 300–850 руб.) [vois-profile.md, vois-research.md].

| Показатель | Значение |
|------------|----------|
| Ценовой сегмент | Масс-маркет |
| Диапазон цен | 300–2 000 руб. |
| Средний чек | 500–1 000 руб. [vois-research.md] |
| AOV (средний заказ) | ~700 руб. (оценка по данным WB) |
| Частота покупок | 4–8 раз в год (цель: увеличить до 8) [vois-research.md] |
| LTV | 3 000–8 000 руб. [vois-research.md] |
| ROS (рентабельность продаж) | 30,6% (2024) [vois-research.md] |

#### 1.2 Выбранная стратегия: **конкурентное ценообразование с элементами ценности**

Обоснование выбора:

1. **Рынок масс-маркет высококонкурентен.** Конкуренты: MIXIT, Natura Siberica, CeraVe, La Roche-Posay, Чистая Линия. VOIS не может позиционироваться по цене как самый дешёвый (Чистая Линия дешевле), но и не претендует на премиум [assignment-4-part2.md].

2. **Целевая аудитория (женщины 18–35, минималистки/практичные)** ценит соотношение «цена/качество» [assignment-5.md].

3. **Структура цены** (по лекции 5): основные элементы — себестоимость и прибыль. VOIS имеет высокую ROS (30,6%), что даёт пространство для манёвра.

4. **Поле ценового решения** ограничено [lecture-5.md]:
   * **Верхняя граница (А):** воспринимаемая ценность — для VOIS выше среднего благодаря INCI-декодеру, версионированию и отзывам (1,6 млн+, рейтинг 4,8).
   * **Нижняя граница (В):** себестоимость — переменные затраты на производство в Китае + логистика + комиссии маркетплейсов (до 24–28%).
   * **Ориентир (С):** цены конкурентов — MIXIT (~400–1 200 руб.), Natura Siberica (~300–1 500 руб.), CeraVe (~800–2 500 руб.).

#### 1.3 Эластичность спроса

Спрос на косметику масс-маркет **эластичный** ($E > 1$): потребители чувствительны к цене [lecture-5.md].

Особенности для VOIS:
* Высокая лояльность (1,6 млн отзывов, LTV 3 000–8 000 руб.) снижает эластичность для повторных покупок.
* Для новых клиентов спрос более эластичен — решение принимается на основе отзывов и сравнения цен.

### 2. Метод ценообразования

#### 2.1 Основной метод: **ориентированный на конкуренцию — «следование за лидером»**

Обоснование:

| Критерий | Оценка |
|----------|--------|
| Конкурентная среда | Высокая (MIXIT, Natura Siberica, СТМ маркетплейсов) |
| Дифференциация | Средняя (INCI-декодер, версионирование — уникальные, но копируемые преимущества) |
| Контроль над ценой | Ограниченный (маркетплейсы диктуют динамику цен и скидок) |
| Целевая аудитория | Ценит соотношение цена/качество, сравнивает с конкурентами |

VOIS устанавливает цены на уровне или чуть ниже ключевых конкурентов (MIXIT, Natura Siberica), компенсируя за счёт уникальных атрибутов (INCI-декодер, отзывы).

#### 2.2 Вспомогательный метод: **ориентированный на потребителя — на основе ощущаемой ценности**

Применяется для премиальной линейки (если запускается) и продуктов с уникальными свойствами:
* Маска Total Repair — цена ~380 руб. за 350 мл при 766 472 отзывах и рейтинге 4,9. Ощущаемая ценность выше цены (восприятие «лучшее соотношение цена/объём») [assignment-4-part2.md].
* VOIS Community (подписка 1 990–3 490 руб./мес.) — цена основана на воспринимаемой ценности эксклюзивного контента и мероприятий [vois-research.md].

#### 2.3 Метод НЕ применяемый: ориентированный на издержки

Причина: VOIS — бренд с сильным маркетингом и эмоциональной связью с аудиторией. Цена формируется рынком и конкурентами, а не только себестоимостью. ROS 30,6% показывает, что текущая цена значительно превышает издержки, то есть есть запас для стратегических скидок.

### 3. Оценка уровня цен на товары

#### 3.1 Ценовая матрица по категориям

| Категория | Продукт | Цена, руб. | Позиционирование | Обоснование |
|-----------|---------|------------|-------------------|-------------|
| **Волосы (флагман)** | Маска Total Repair, 350 мл | ~380 | **Ценность-лидер** | Лучшее соотношение цена/объём в категории, 766K отзывов [assignment-4-part2.md] |
| **Лицо (базовый уход)** | Гель для умывания | ~350–450 | **Конкурентная цена** | На уровне MIXIT, ниже CeraVe |
| **Лицо (интенсивный)** | Сыворотка от акне | ~550–750 | **Ценность-лидер** | Рост 1554% г/г, уникальная формула [vois-research.md] |
| **Глаза** | Тканевые патчи (набор) | ~450–650 | **Конкурентная цена** | На уровне рынка, высокие отзывы |
| **Тело** | Крем-баттер | ~400–550 | **Конкурентная цена** | Базовый уход, невысокая дифференциация |
| **Наборы** | Beauty box (5 бестселлеров) | ~1 200–1 500 | **Психологическая цена** | Восприятие выгоды от покупки набора |
| **VOIS Community** | Подписка Standard | 3 490/мес | **Ценностное ценообразование** | Ощущаемая ценность эксклюзивного доступа [vois-research.md] |

#### 3.2 Анализ безубыточности (CVP Analysis)

**Пример: Маска Total Repair**

| Параметр | Значение (оценка) |
|------------|-------------------|
| Цена (P) | 380 руб. |
| Переменные затраты (AVC) | ~120 руб. (производство + логистика + комиссия MP) |
| Маржинальный доход (P – AVC) | 260 руб. |
| Постоянные затраты (FC) на линейку | ~50 млн руб./год (маркетинг, разработка, сертификация) |
| **Точка безубыточности** | $Q_{bp} = \frac{50\,000\,000}{260} \approx 192\,308$ шт./год |
| Фактические продажи (2024) | ~1 500 000+ шт. (579 млн руб. / ~380 руб.) |

**Вывод:** продукт значительно превышает точку безубыточности. Запас финансовой прочности позволяет снижать цену для стимулирования или повышать инвестиции в маркетинг.

#### 3.3 Рекомендации по корректировке цен

| Направление | Рекомендация | Обоснование |
|-------------|-------------|-------------|
| **Повышение цен** | На 5–10% для продуктов с низкой эластичностью (Total Repair, патчи) | Высокая лояльность, лучшие отзывы на рынке |
| **Дискриминационное ценообразование** | Скидки 10–15% для подписчиков VOIS Community | Лекция 5: разные цены для разных групп по времени/членству [lecture-5.md] |
| **Психологические цены** | 399 руб. вместо 400, 1 990 руб. вместо 2 000 | Восприятие «до 400» или «до 2 000» [lecture-5.md] |
| **Премиальная линейка** | 1 500–2 500 руб. (пептиды, биотехнологии) | Выход в сегмент с более высокой маржинальностью |
