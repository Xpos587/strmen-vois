# Задание 5

**Лекция:** Стратегический менеджмент (лекции 3-4)
**Источник:** `lectures/lecture-4/Лекция 3-4 СтрМен.pdf`

## Задание

1. Провести сегментирование рынка, выделив наиболее значимые параметры.
2. Обосновать выбор целевого сегмента.
3. Составить профиль клиента / клиентов.

## Методические указания

**Параметры сегментации:** географические, демографические, психографические, поведенческие.

**Варианты охвата рынка:**
- **Концентрированная:** выход только на один сегмент.
- **Дифференцированная:** несколько сегментов рынка.
- **Недифференцированная:** обслуживание всех сегментов одним ассортиментом.
