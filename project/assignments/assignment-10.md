### 2. SWOT-стратегии

#### 2.1 SO-стратегии (Силы + Возможности)

| Стратегия | Комбинация | Суть |
|-----------|------------|------|
| **SO1** | S1 (лидерство на МП) + O1 (рост российских брендов) | Усилить позиционирование как флагмана российского бьюти, запустить кампанию «Выбирай своё» |
| **SO2** | S2 (Content Lab) + O8 (цифровая экосистема) | Интегрировать Content Lab в VOIS Community: эксклюзивный контент для подписчиков |
| **SO3** | S3 (1.6 млн отзывов (оценка, прямой источник не найден)) + O5 (премиализация) | Использовать отзывы как социальное доказательство для запуска премиальной линейки |
| **SO4** | S4 (высокая рентабельность) + O3 (офлайн) | Инвестировать прибыль в развитие офлайн-каналов и собственной розницы |
| **SO5** | S5 (INCI-декодер) + O7 (экология) | Запустить «зелёную» линейку с экологичной упаковкой и INCI 2.0 |

#### 2.2 ST-стратегии (Силы + Угрозы)

| Стратегия | Комбинация | Суть |
|-----------|------------|------|
| **ST1** | S2 (Content Lab) + T5 (негативный PR) | Использовать штатных креаторов для быстрого реагирования на негатив и нейтрализации |
| **ST2** | S3 (отзывы) + T2 (СТМ конкуренция) | Усилить программу отзывов как барьер для СТМ — социальное доказательство невозможно скопировать |
| **ST3** | S7 (Total Repair) + T3 (подделки) | Защита бренда через патентование формул и юридические действия (56 дел как истец) [vois-research.md] |
| **ST4** | S4 (рентабельность) + T1 (рост комиссий) | Использовать высокую ROS (~23,2%) для поглощения роста комиссий без повышения цен на 1–2 года |
| **ST5** | S6 (data-driven) + T6 (волатильность) | Прогнозировать спрос на основе данных для оптимизации запасов и снижения рисков |

#### 2.3 WO-стратегии (Слабости + Возможности)

| Стратегия | Комбинация | Суть |
|-----------|------------|------|
| **WO1** | W1 (зависимость от МП) + O3 (офлайн) | Развивать офлайн-канал через Золотое Яблоко и собственные точки для диверсификации |
| **WO2** | W3 (нет e-commerce) + O8 (экосистема) | Запустить собственный e-commerce в рамках VOIS Community для D2C-продаж |
| **WO3** | W6 (узкая линейка) + O6 (декоративка) | Расширить ассортимент в декоративную косметику через партнёрства или собственную разработку |
| **WO4** | W5 (скепсис) + O4 (экспансия) | Использовать международное присутствие как доказательство качества и легитимности |
| **WO5** | W2 (Китай) + O7 (экология) | Пересмотреть поставщиков с учётом экологических стандартов |

#### 2.4 WT-стратегии (Слабости + Угрозы)

| Стратегия | Комбинация | Суть |
|-----------|------------|------|
| **WT1** | W1 (зависимость от МП) + T1 (рост комиссий) + T2 (СТМ) | Срочная диверсификация: развитие Ozon, запуск D2C, поиск альтернативных каналов |
| **WT2** | W2 (Китай) + T6 (валютные риски) | Хеджирование валютных рисков, поиск локальных поставщиков сырья |
| **WT3** | W4 (мало офлайна) + T4 (возврат зарубежных брендов) | Ускорить выход в офлайн-ритейл для конкуренции с возвращающимися брендами |
| **WT4** | W5 (скепсис) + T5 (негативный PR) | Создать кризисный коммуникационный план с быстрым ответом через Content Lab |
| **WT5** | W7 (сходство упаковки) + T3 (подделки) | Усилить защиту дизайна через регистрацию товарных знаков (55–59 зарегистрировано) [vois-research.md] |

### 3. Приоритетная матрица стратегий

| Приоритет | Стратегия | Категория | Обоснование приоритета |
|:-:|:-:|:-:|------------------------|
| 1 | SO1, ST4, WT1 | Диверсификация каналов | Критическая зависимость от МП + рост комиссий |
| 2 | SO3, WO3 | Продуктовое развитие | Премиализация и расширение линейки |
| 3 | ST1, WT4 | Репутационная защита | Защита от негатива и подделок |
| 4 | SO5, WO5 | Экологичность | Долгосрочный тренд |
| 5 | SO4, WO2 | Инвестиции в инфраструктуру | Финансовая мощь позволяет |
